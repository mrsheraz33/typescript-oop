# TypeScript Abstract Classes

This folder contains TypeScript examples for learning Abstract Classes.

## Topics Covered

* Abstract Classes
* Abstract Methods
* Inheritance
* extends
* Method Implementation

## Examples

### Vehicle Class

* Create an abstract parent class
* Define a common method (`honk()`)
* Declare an abstract method (`startEngine()`)

### Car Class

* Extend the `Vehicle` class
* Implement the `startEngine()` method
* Start the car using a button

### Bike Class

* Extend the `Vehicle` class
* Implement the `startEngine()` method
* Start the bike using kick/self

# TypeScript Interfaces

This folder contains TypeScript examples for learning Interfaces.

## Topics Covered

* Interfaces
* implements
* Interface Properties
* Interface Methods
* Method Implementation

## Examples

### SmartphoneApp Interface

* Define a common structure for smartphone applications
* Declare properties and methods
* Implement the interface in multiple classes

### WhatsApp Class

* Implement the `SmartphoneApp` interface
* Open and close the application

### YouTube Class

* Implement the `SmartphoneApp` interface
* Provide its own implementation of interface methods

### SearchSystem Interface

* Define a search functionality
* Implement the interface using the `GoogleSearch` class
* Demonstrate method parameters with an optional argument

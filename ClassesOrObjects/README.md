# TypeScript Classes

This folder contains TypeScript examples for learning Classes and Objects.

## Topics Covered

* Classes
* Objects
* Constructors
* Class Properties
* Class Methods

## Examples

### BankAccount Class

* Create objects using constructors
* Store account holder information
* Display account balance

### Car Class

* Store car details
* Access properties
* Call class methods

# TypeScript Abstraction

This folder contains TypeScript examples for learning Abstraction.

## Topics Covered

* Abstraction
* Abstract Classes
* Data Hiding
* Method Encapsulation
* Inheritance

## Examples

### CoffeeMachine Class

* Create an abstract base class
* Hide internal implementation details
* Perform coffee-making steps internally
* Expose a single public method to make coffee

### PremiumCoffeeMaker Class

* Extend the `CoffeeMachine` class
* Inherit the coffee-making functionality
* Use the abstract class without exposing internal methods

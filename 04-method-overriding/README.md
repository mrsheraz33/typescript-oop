# TypeScript Method Overriding

This folder contains TypeScript examples for learning Method Overriding.

## Topics Covered

* Method Overriding
* Inheritance
* extends
* super()
* Public
* Protected

## Examples

### Vehicle Class

* Create a parent class
* Define common properties and methods
* Start the vehicle engine
* Default `honk()` method

### Car Class

* Extend the `Vehicle` class
* Override the `honk()` method
* Access inherited members
* Display vehicle speed

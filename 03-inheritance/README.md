# TypeScript Inheritance

This folder contains TypeScript examples for learning Inheritance using the `extends` keyword.

## Topics Covered

* Inheritance
* extends
* Parent Class
* Child Class
* super()
* Public
* Private
* Protected

## Examples

### Employee Class

* Create a parent class
* Define public, private, and protected members
* Access employee details
* Calculate employee bonus

### Manager Class

* Extend the `Employee` class
* Use the `super()` constructor
* Access inherited public and protected members
* Add manager-specific properties and methods

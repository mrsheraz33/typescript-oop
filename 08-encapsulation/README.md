# TypeScript Encapsulation

This folder contains TypeScript examples for learning Encapsulation.

## Topics Covered

* Encapsulation
* Private Members
* Data Hiding
* Getter Methods
* Object-Oriented Programming (OOP)

## Examples

### SecureAccount Class

* Store balance as a private property
* Access balance using a getter method
* Deposit money securely
* Prevent invalid balance updates

# TypeScript Polymorphism

This folder contains TypeScript examples for learning Polymorphism.

## Topics Covered

* Polymorphism
* Method Overriding
* Runtime Polymorphism
* Inheritance
* Dynamic Method Dispatch

## Examples

### Animal Class

* Create a base class
* Define a common `makeSound()` method

### Dog Class

* Extend the `Animal` class
* Override the `makeSound()` method

### Cat Class

* Extend the `Animal` class
* Override the `makeSound()` method

### Runtime Polymorphism

* Store different objects in an `Animal[]` array
* Call the same method on different objects
* Execute different implementations at runtime
